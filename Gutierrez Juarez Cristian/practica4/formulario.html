Todos los formularios que se manejan en la web utilizan metodos para el envio o respuesta de la informacion, los mas comunes son:


GET: Este metodo se encarga de enviar la informacion por la url o salida estandar de navegador, esto propicia que se pueda visualizar toda la informacion del formulario.
POST:Este metodo se encarga de enviar la informacion por la url pero de forma oculta, lo que ofrece cierto nivel de seguridad
